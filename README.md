# DIVA 2.0 — Entretien diagnostique interactif TDAH adulte

Outil interactif d'aide à la passation de la DIVA 2.0 (Kooij & Francken, 2010 — DIVA Foundation).

⚠️ **Cet outil ne remplace pas le jugement clinique d'un professionnel de santé qualifié.**

## Fonctionnalités

- 18 critères DSM-IV avec exemples cochables (âge adulte + enfance)
- Auto-détection : « Oui » s'active quand la majorité des exemples est cochée
- Retentissement dans 5 domaines (adulte + enfance) avec bouton Oui/Non par domaine
- Critères B, C/D et E du DSM-IV
- Score automatique et sous-type (combiné / inattentif / hyperactif-impulsif)
- Barre de progression globale
- Sauvegarde automatique (localStorage)

## Déploiement

```bash
npm install
npm run dev
```

Vercel : connecter le repo GitHub, Vercel détecte Next.js automatiquement.
